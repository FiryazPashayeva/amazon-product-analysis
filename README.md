# Amazon Product Analysis

## Project Overview

This project analyzes Amazon product data using Python and Tableau. The dataset was obtained from Kaggle and cleaned using Pandas before creating an interactive dashboard in Tableau Public.

## Dataset

- **Source:** Kaggle
- **Dataset:** Amazon Product Dataset

## Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Tableau Public

## Data Cleaning

- Checked missing values
- Removed invalid records
- Converted data types
- Cleaned prices, ratings, and discount percentages
- Performed normality analysis
- Used median imputation for missing values
- Created new features (`Savings` and `Main_Category`)
- Exported a cleaned dataset for visualization

## Dashboard

The dashboard includes:

- Average Rating by Category
- Average Discount by Category
- Price vs Rating (Scatter Plot)
- Discount Distribution (Histogram)
- Top Reviewed Products
